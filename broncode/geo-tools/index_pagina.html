Wat is het?

Gereedschap om met geogeafische informatie te kunnen omgaan die conform STOP in GML is vastgelegd. Het gaat met name om de geografische informatieobjecten (GIO's). Laat ook zien hoe de randvoorwaarden voor het muteren van GIO's ingevuld kunnen worden.

Wat zit er in?

Twee webpagina's om aanvullende bestanden te maken die kunnen helpen om een GIO in GIS software (onder andere QGIS) te tonen:

  • GFS bestand voor GML die in STOP gebruikt wordt voor een GIO, gebiedsmarkering of effectgebied.
  • SLD bestand voor een symbolisatie

Verder een aantal Python scripts die aan software leveranciers demonstreren hoe een GIO-wijziging bepaald en getoond kan worden. De scripts zijn online uit te voeren:

  • Bepaling van een GIO-wijziging uit twee versies van een GIO.
  • Tonen van een GIO-wijziging die de geo-renvooi voor een GIO-wijziging toont.

De scripts sluiten aan bij de beschrijving in STOP.
